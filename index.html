Malaysian Car Listings Analysis

Project Aim

The aim of this project is to analyze and visualize data from the Kaggle dataset using advanced data visualization techniques. By creating an interactive dashboard, we aim to provide insights into the trends, patterns, and key metrics present in the dataset. The dashboard will help users better understand the significant factors and relationships within the data, facilitating informed decision-making. The visualizations will be designed to present the data in a clear, engaging, and organized manner, promoting a deeper understanding of the dataset's contents and supporting data-driven decisions.

Objectives

  • To gather the data regarding the information of Malaysian Car Listings.
  • To analyze the data to get the patterns of a Malaysian Car Listings.
  • To develop a dashboard visualization which is to provide the information that associated with the Malaysia Car Listing to help those who want to buy a car.

Methodology

The dataset contains records of 920 data points with 18 columns, including details on car specifications, seller information, and pricing in Malaysia. The data was sourced from Oto.my and analyzed using Microsoft Power BI.

Findings

This project has created a user-friendly car listing dashboard for the Malaysian automotive market. The dashboard provides valuable insights into car prices, popular brands, and trends, helping buyers, sellers, and policymakers make informed decisions.

Power BI Dashboard

Conclusion

In conclusion, this project has created a user-friendly car listing dashboard for the Malaysian automotive market. This dashboard provides valuable insights into car prices, popular brands, and trends, helping buyers, sellers, and policymakers make informed decisions. By utilizing data from Oto.my, the dashboard allows users to easily compare different car listings, making the car buying process simpler and more transparent. Looking ahead, enhancing the dashboard with real-time data will further improve its usefulness and provide deeper insights into the market. Overall, this project aims to enhance customer satisfaction and streamline the car buying experience in Malaysia, making it a valuable resource for everyone involved.
