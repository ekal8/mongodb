Malaysian Car Listings Analysis

Project Aim

The aim of this project is to analyze and visualize data from the Kaggle dataset using advanced data visualization techniques. By creating an interactive dashboard, we aim to provide insights into the trends, patterns, and key metrics present in the dataset.

Objectives

  • Gather data regarding Malaysian car listings.
  • Analyze the data to identify patterns in Malaysian car listings.
  • Develop a dashboard visualization to provide information associated with Malaysian car listings.

Methodology

The dataset contains records of 920 data points with 18 columns, including details on car specifications, seller information, and pricing in Malaysia. The data was sourced from Oto.my and analyzed using Microsoft Power BI.

Findings

This project has created a user-friendly car listing dashboard for the Malaysian automotive market. The dashboard provides valuable insights into car prices, popular brands, and trends, helping buyers, sellers, and policymakers make informed decisions.

Power BI Dashboard

Conclusion

By utilizing data from Oto.my, the dashboard allows users to easily compare different car listings, making the car buying process simpler and more transparent. Enhancing the dashboard with real-time data will further improve its usefulness and provide deeper insights into the market.
